Programmatic Graphics Extraction Strategy

Claude prompts alone are failing to capture accurate bounding boxes. We need robust logic to place S3 assets and separators with 95%+ accuracy.

A

Geometric / Spatial Heuristics

Analyze high-accuracy text blocks to infer where separators must exist. (e.g., vertical gaps between Category Headers).

Pros

  • Fast & Predictable
  • No image noise

Cons

  • May invent separators
  • No logo detection
B

Computer Vision (OpenCV)

Use Hough Lines, Contours, and Template Matching directly on the pixels to detect physical shapes.

Pros

  • Finds actual pixels
  • Great for lines

Cons

  • Noise sensitive
  • No semantic context
C

Hybrid: CV + Semantic Filtering

Use OpenCV to find shapes, then use text bounding boxes to validate and label them semantically.

Pros

  • Highest potential (95%+)
  • Context-aware reality

Cons

  • Higher complexity
  • More tuning required
